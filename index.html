My Story

From using tools
to wondering how they're built.

I started in digital marketing, running campaigns and studying analytics. The more I used the tools, the more I found myself asking different questions: how does this tool work? Why is it designed this way? Could it be simpler?

That curiosity pulled me toward software engineering. I wanted to understand what happens behind the screen. So I started learning to code. Python first. Then Git. Then building small projects late at night when I should have been sleeping.

I'm not a professional yet. But I'm someone who notices when things could work better and wants to learn how to build them that way.

Now I'm applying for Software Engineering because I want to understand the fundamentals: how code thinks, how systems scale, and how technology is truly built.
